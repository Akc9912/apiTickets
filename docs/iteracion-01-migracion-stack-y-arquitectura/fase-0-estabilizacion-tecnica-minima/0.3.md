# 0.3 - Congelar contrato API baseline

## Objetivo

Congelar el contrato API actual para poder comparar regresiones funcionales durante la migracion.

## Razon Tecnica

La iteracion modifica capas, seguridad y modelo de datos. Sin contrato baseline no se puede validar compatibilidad ni cambios intencionales.

## Comandos Exactos

```bash
mkdir -p docs/iteracion-01-migracion-stack-y-arquitectura/fase-0-estabilizacion-tecnica-minima/evidencia
curl -s http://localhost:8080/v3/api-docs > docs/iteracion-01-migracion-stack-y-arquitectura/fase-0-estabilizacion-tecnica-minima/evidencia/openapi-baseline.json
cp endpoints.md docs/iteracion-01-migracion-stack-y-arquitectura/fase-0-estabilizacion-tecnica-minima/evidencia/endpoints-baseline.md
```
