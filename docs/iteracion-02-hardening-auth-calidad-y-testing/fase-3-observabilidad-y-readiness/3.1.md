# 3.1 - Activar metricas y health checks operativos

## Objetivo

Activar metricas basicas y health checks para monitoreo inicial post-MVP.

## Razon Tecnica

Sin visibilidad basica de salud y rendimiento no se puede operar con confianza ni detectar degradaciones tempranas.

## Comandos Exactos

```bash
curl -s http://localhost:8080/actuator/health
curl -s http://localhost:8080/actuator/metrics
./mvnw -DskipTests compile
```

## Ejemplo De Codigo (Si Aplica)

```txt
metricas minimas:
- http.server.requests
- jvm.memory.used
- process.cpu.usage
```

## Verificacion

1. /actuator/health responde status UP en entorno estable.
2. /actuator/metrics lista metricas minimas habilitadas.
3. Se documenta lista de indicadores base a observar.

## Resultado Esperado

Base de monitoreo operativo disponible para las iteraciones funcionales siguientes.

- [ ] objetivo cumplido
- [ ] verificacion ejecutada

## Troubleshooting Breve (Si Aplica)

- metricas faltantes -> verificar dependencias y configuracion de management
